# 🎈 Pocket Life - 당신의 일상을 가볍고 스마트하게

안녕하세요! **Pocket Life** 저장소에 오신 것을 환영합니다. 👋
이 프로젝트는 복잡한 일상을 심플하게 관리할 수 있도록 돕는 **올인원 라이프스타일 매니지먼트 웹 애플리케이션**입니다.

가계부 부터 식단, 일정 관리까지, 흩어져 있던 일상을 한곳에 모았습니다.
사용자 친화적인 인터페이스와 직관적인 데이터 시각화로 여러분의 '포켓 라이프'를 시작해보세요!

---

## 🚀 기술 스택 (Tech Stack)

프론트엔드 본연의 즐거움에 집중하여 개발했습니다.

| Category | Technology |
|----------|------------|
| **Core** | React, Vite |
| **Logic** | JavaScript (ES6+) |
| **Network** | Axios |
| **Data Viz** | Recharts, Chart.js |
| **Data** | 공공데이터 API (Public Data API) |
| **UI Libs** | React Datepicker |

---

## ✨ 주요 기능 미리보기

### 1. 한눈에 들어오는 대시보드 (Dashboard)
현재 상태를 직관적으로 파악할 수 있는 메인 대시보드입니다. 
한눈에 모든정보의 요약내용을 파악 및 관리할 수 있습니다.

![Dashboard Preview](./src/image/gif/Dashboard.gif)

### 2. 똑똑한 가계부 관리 (Ledger)
수입과 지출을 캘린더와 리스트로 간편하게 관리하세요. '어디에 돈을 썼지?' 고민할 필요 없이, 통계 그래프가 당신의 소비 패턴을 분석해 드립니다.

![Ledger Preview](./src/image/gif/Ledger.gif)

### 3. 건강한 식단 기록 (Meal)
오늘 무엇을 먹었는지 기록하고, 칼로리와 영양 성분을 체크해보세요.

![Meal Preview](./src/image/gif/Meal.gif)

### 4. 편리한 장바구니 (Shopping Cart)
사야 할 물건들을 잊지 않도록 메모하고 관리합니다. 체크리스트 기능으로 알뜰한 쇼핑을 도와드려요.

![Cart Preview](./src/image/gif/Cart.gif)

### 5. 빈틈없는 일정 관리 (Schedule)
중요한 약속과 할 일을 놓치지 마세요. 공공데이터 API를 통해 공휴일 정보를 확인할 수 있습니다.

![Schedule Preview](./src/image/gif/Schedule.gif)

### 6. 간편하고 안전한 시작 (Authentication)
회원가입과 로그인은 빠르고 부드럽게. 사용자 경험을 최우선으로 고려한 진입 화면입니다.

![Auth Preview](./src/image/gif/Signup_Login.gif)

## 🏃‍♂️ 시작하기 (Getting Started)

이 프로젝트를 로컬 환경에서 실행해보고 싶으신가요? 아래 명령어를 차례대로 입력해주세요.

```bash
# 1. 저장소 클론
git clone https://github.com/your-username/pocket-life-frontend.git

# 2. 프로젝트 폴더로 이동
cd pocket-life-frontend

# 3. 패키지 설치
npm install

# 4. 개발 서버 실행
npm run dev
```
