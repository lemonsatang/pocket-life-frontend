# 🎈 Pocket Life - 당신의 일상을 가볍고 스마트하게

안녕하세요! **Pocket Life** 저장소에 오신 것을 환영합니다. 👋
이 프로젝트는 복잡한 일상을 심플하게 관리할 수 있도록 돕는 **올인원 라이프스타일 매니지먼트 웹 애플리케이션**입니다.

가계부 부터 식단, 일정 관리까지, 흩어져 있던 일상을 한곳에 모았습니다.
사용자 친화적인 인터페이스와 직관적인 데이터 시각화로 여러분의 '포켓 라이프'를 시작해보세요!

---

## 🚀 기술 스택 (Tech Stack)

프론트엔드 본연의 즐거움에 집중하여 개발했습니다.

| Category | Technology |
|----------|------------|
| **Core** | React, Vite |
| **Logic** | JavaScript (ES6+) |
| **Network** | Axios |
| **Data Viz** | Recharts, Chart.js |
| **Data** | 공공데이터 API (Public Data API) |
| **UI Libs** | React Datepicker |

## 🏃‍♂️ 시작하기 (Getting Started)

이 프로젝트를 로컬 환경에서 실행해보고 싶으신가요? 아래 명령어를 차례대로 입력해주세요.

```bash
# 1. 저장소 클론
git clone [https://github.com/your-username/pocket-life-frontend.git](https://github.com/your-username/pocket-life-frontend.git)

# 2. 프로젝트 폴더로 이동
cd pocket-life-frontend

# 3. 패키지 설치
npm install

# 4. 개발 서버 실행
npm run dev
