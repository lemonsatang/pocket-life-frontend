# 통계 페이지 파이 차트 변경 계획

## 변경 목표
통계 페이지의 방사형 차트를 원형(파이) 차트 3개로 변경

## UI 구조
- **CHART 01**: 식단관리 (기존 dietScore 사용)
- **CHART 02**: 소비율 (기존 consumptionScore 사용)
- **CHART 03**: 지출율 (새로 계산 필요)
  - 세부사항: 카테고리별 지출 상위 2개 표시

## 구현 계획

### 1. 차트 라이브러리 변경
- `Radar` → `Pie` 차트로 변경
- Chart.js의 `ArcElement` 등록 필요
- 각 파이 차트 위에 "CHART 01", "CHART 02", "CHART 03" 라벨 표시

### 2. 데이터 구조
- CHART 01: 식단관리 점수 (0-100)
- CHART 02: 소비율 점수 (0-100)
- CHART 03: 지출율 점수 (0-100)
- CHART 03 세부사항: 카테고리별 지출 금액 상위 2개

### 3. 레이아웃
- 파이 차트 3개를 가로로 배치
- 각 차트 아래에 세부사항 표시
- CHART 03만 세부사항(카테고리별 지출 상위 2개) 표시

## 참고사항
- 기존 Radar 차트 관련 코드 제거
- Chart.js Pie 차트 옵션 설정 필요
- 반응형 디자인 고려
