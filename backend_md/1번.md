# 통계 페이지 파이 차트 변경 관련 문의

## 요청 사항
통계 페이지의 방사형 차트를 원형(파이) 차트 3개로 변경 예정입니다.
이를 위해 다음 정보 확인이 필요합니다.

## 필요한 데이터 확인

### 1. 지출율 계산 방식
현재 코드에서는 소비율과 저축률만 계산하고 있습니다.
- **지출율** 계산 방식 확인이 필요합니다.
  - 제안: `(총 지출 / 총 수입) * 100` 또는 다른 계산 방식인지?
  - 수입이 0인 경우 지출율 표시 방법?

### 2. 카테고리별 지출 통계
CHART 03의 세부사항에 "가장 지출이 집중되는 목록 2개"를 표시하려고 합니다.
- 현재 `/api/tx` API로 전체 거래 내역을 받아서 프론트에서 카테고리별 집계가 가능한지?
- 또는 카테고리별 지출 통계를 제공하는 별도 API가 있는지?

### 3. 거래 데이터 필드명 확인
- 거래 날짜 필드명: `date`, `txDate`, `transactionDate` 중 어느 것을 사용하나요?
- 카테고리 필드명: `category`가 맞나요?

## 참고
현재 `/api/tx` API로 전체 거래 내역을 받아서 프론트에서 필터링하고 있습니다.
월별 필터링을 위해 날짜 필드명 확인이 필요합니다.
