# 통계 페이지 4분할 차트 및 주/월 단위 통계 관련 문의

## 요청 사항
통계 페이지를 4개 차트로 변경하고, 주 단위/월 단위 선택 기능을 추가하려고 합니다.

## 필요한 API 확인

### 1. 주 단위 통계 API
- 현재 월 단위 통계만 지원하는 것으로 보입니다.
- **주 단위 통계를 위한 API가 있는지 확인이 필요합니다.**
  - 예: `/api/stats/meal?startDate=2026-01-15&endDate=2026-01-21` 형식
  - 또는 `/api/stats/meal?week=2026-W03` 형식

### 2. 기간 범위 조회 API
다음 API들이 기간 범위(startDate, endDate)로 조회 가능한지 확인이 필요합니다:
- `/api/stats/meal` - 식단 통계
- `/api/stats/cart` - 장바구니 통계
- `/api/todo/getList` - 일정 목록
- `/api/tx` - 거래 내역
- `/api/tx/summary` - 거래 요약

### 3. 일정 통계 API
- 현재 `/api/todo/getList`는 특정 날짜의 일정만 조회하는 것으로 보입니다.
- **기간 범위 내의 전체 일정과 완료된 일정 수를 조회할 수 있는 API가 있는지?**
  - 예: `/api/todo/stats?startDate=2026-01-15&endDate=2026-01-21`

### 4. 장바구니 통계 API
- 현재 `/api/stats/cart`는 특정 날짜만 조회하는 것으로 보입니다.
- **기간 범위 내의 장바구니 통계를 조회할 수 있는지?**
  - 전체 등록 항목 수와 구매 완료 항목 수를 기간별로 집계 가능한지?

## 데이터 구조 확인

### 소비율 차트 변경
- 기존: 총 지출 / 기준값
- 변경: 카테고리별 지출 비율을 파이 차트로 표시
- 각 카테고리별 지출 금액과 비율(%)이 필요합니다.

## 참고
- 주 단위: 월요일~일요일 기준으로 계산 예정
- 이전 기간 대비 증감률 계산을 위해 현재 기간과 이전 기간 데이터가 모두 필요합니다.
